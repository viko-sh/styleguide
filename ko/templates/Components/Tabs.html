Documentations
  1. css & knockout component
  2. optional:
    1. tabs with check & text only
    2. tabs with check & icon & text
  3. knockout component element name: 'ko.style--tabs'
  4. parameters:
    • tabs - ( enum array with object )
      Example Tab Object
      • text ( Required )
      • selected ( Required )
      • icon ( Optional )
      • socialNetwork ( Optional )
    • multiple - true if user can select multiple tabs , false by default
    • instance - if you need the instance of the component once its loaded
    • onChange - callback will return the selected tab, or selected tabs if multiple tabs selection is enabled
    • border - if to set border around the component
<ko.style--tabs class="margin-left--40 margin-right--40 border-bottom-dashed"
    params="tabs: _TABS,
            border: false,
            onChange: onSelectedTabs.bind($component),
            multiple: true,
            instance: tabsInstance">
</ko.style--tabs>
