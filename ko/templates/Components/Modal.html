Modal

Used In: Topics builder
Documentations
  1. css & knockout component
  2. parameters:
    1. beforeOpen - callback function before modal is opened, reutrn false will disable the modal
    2. closeFunction - before close function, can run stuff and return false to prevent closing.
    3. applyFunction - before close function, can run stuff and return false to prevent closing.
    4. footerErrorMsg - most be observable or nothing at all !
    5. applyDisabledWhen - most be observable or nothing at all !
    6. element - ID reference to the template element the modal will display
    7. headerButtons - list of icons with callback function to display in the header.
      • icon OR class
      • callback
      • hover
      • tooltip
      • disabled
    8. hideFooter - set this to true if you dont want to display footer buttons
    9. model - option to also have a reference to the modal JS
    10. css - modal wrapper css classes we want to add
    11. buttonClass - button class we want to add, this button click will open the modal
    12. buttonText - button text
    13. buttonTooltip - button tooltip text
 <ko.modal class="modalComponent"
   params='
        element: "unique-modal-id-elementForModal22",
        model: $data.model(),
        closeFunction: $data.closeClickFunction.bind($data),
        applyFunction: $data.applyClickFunctionReturnFalse.bind($data),
        beforeOpen:  $data.beforeOpenFunction,
        footerErrorMsg: $data.hasErrorsTextMsgModal2,
        applyDisabledWhen: $data.applyDisabledWhenModal3,
        headerButtons: $data.headerButtons3,
        hideFooter: true,
        buttonClass: "button btn-secondary",
        buttonText: "Open Modal Example #3 - apply disabled",
        buttonTooltip: "Button Tooltip",
        css: ""'>
 </ko.modal>
