  1. css & knockout binding wrapper for foundation date picker (fdatepicker)
  2. create span wrapper with class 'date-picker' and 'background-tracx-white'
  3. create icon inside the span wrapper 'tr-icon-calendar' with whatever color you want
  4. create input with type text
  5. data bind fdatepicker, with the following options:
    • date - current date ( TODAY ) , will be the default when the widget starts
    • datepickerOptions:
      • startDate - the min date the user can select
      • initialDate - the first date the user will see by default ( usually current date )
      • isUsingMoment - true/false flag ( if not converting with moment , will use 'convertJSDateToSqlString' function from utils.js
      • dateFormat - when using moment change to this format, if no moment is used, will convert using 'convertJSDateToSqlString' function
      • changeCallback - once the date is changed, will call this function with the new date ( date format depends on moment usage or not ) 
fdatepicker: { date: $data.startDate,
                        datepickerOptions: {
                            startDate: $data.minDate,
                            initialDate: $data.startDate(),
                            dateFormat: $data.dateFormat,
                            isUsingMoment:     true
                         } }
