Documentations
  1. css & knockout component
  2. type:
    1. 0 - default - not necessary
    2. 1 - icons list type.
    3. 2 - checkbox from the left
    4. 3 - checkbox from the right
  3. Sizes: small, large, default is medium.
  4. options - list of objects to select from
  5. optionsText - when given will not display the options, but the options[optionsText] field.
  6. optionsValue - declare which field is filled in the selected, if not given, all the object is "selected"
  7. optionsSearch - when given, a search field is added , this can be a key param to filter by. or true for regular strings.
  8. optionsCategory - when given, the list of items will be groupd by this key
    1. category support checkbox
    2. category cant
  9. selected ( when multi is true )
    1. must be observable array
    2. the data will be object if no optionsValue is defined
    3. the data will be the optionsValue field from the selected option
    selected ( when multi is false, or not declared )
    1. must be observable
    2. the data will be object if no optionsValue is defined
    3. the data will be the optionsValue field from the selected option
  10. multi - boolean , true/false
  11. caption
    1. null/undefined/not declared - component will select first value by default on multi=false
    2. multi=true and no caption will result empty select on start
<ko.select class="" params="
    options:                myOptionsObjects,
    optionsText:            'text',
    optionsValue:           'id',
    optionsSearch:          'name',
    optionsCategory:        'category',
    optionsCategoryText:    'category',
    optionsCheckboxLeft:    true,
    optionsCheckboxRight:   true,
    optionsWidth:           '800px',
    optionsConfirm:  {
                        confirmBefore: beforeConfirm.bind($data),
                        confirmAfter: afterConfirm.bind($data),
                        confirmOnlyIf: shouldChange.bind($data),
                        confirmMessage: 'Are you sure? this is UNSAFE MAN!! '
                     },
    selected:               mySelectedObjectOrObjects,
    disable:                true,
    multi:                  true,
    caption:                null
">  </ko.select>
